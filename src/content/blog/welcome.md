Welcome! This site is my personal corner of the internet — a place to share ideas, document projects, and build a living knowledge base.

## What to Expect

- **Blog posts** on software development, tools, and techniques
- **Portfolio** showcasing projects and case studies
- **Knowledge base** articles on topics I explore deeply

## Tech Stack

This site is built with [Astro](https://astro.build), a modern static-site generator that ships zero JavaScript by default. It's deployed to [GitHub Pages](https://pages.github.com) via GitHub Actions.

Key choices:
- **Astro** — fast static output, great Markdown/MDX support, flexible component model
- **GitHub Pages** — free hosting, CI/CD via GitHub Actions
- **MDX** — enhanced Markdown with component support for rich content

## Why Astro?

Astro's "islands architecture" means I can add interactive components exactly where needed without shipping a heavy JavaScript bundle. The content collections API makes it easy to manage blog posts and knowledge base articles with type-safe frontmatter.
